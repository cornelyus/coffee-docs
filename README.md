# Specialty Coffee Docs

A documentation site about specialty coffee — covering brewing methods, water chemistry, processing, tasting, origins, and equipment. Built as a portfolio project to demonstrate Next.js, MDX content pipelines, and (upcoming) AI-powered search.

## Stack

- **Next.js 16.2.2** — App Router, Turbopack
- **Velite 0.3.1** — type-safe MDX content layer
- **Tailwind CSS v4** + `@tailwindcss/typography`
- **rehype-pretty-code** + **Shiki** — syntax highlighting
- **remark-gfm** — GitHub Flavored Markdown

## Content

12 MDX documents under `content/docs/`:

- Getting started & overview
- **Brewing** — pour-over, immersion, espresso
- Processing methods
- Water chemistry
- Tasting & sensory evaluation
- Equipment
- **Origins** — overview, Ethiopia, Colombia

## Development

Velite and Next.js run in parallel during development:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

Velite must run before Next.js (Turbopack doesn't support webpack plugins):

```bash
npm run build
npm start
```

## Project structure

```
content/docs/      # MDX source files
.velite/           # generated by Velite (gitignored)
app/               # Next.js App Router pages and layouts
components/        # UI components
lib/               # content queries and nav helpers
velite.config.ts   # Velite schema and pipeline config
```
